# Proyecto dashboard para prueba en AngularJS + DevExtreme

## Descripción
Se elaboró un mini dashboard desarrollado como prueba técnica frontend usando AngularJS 1.5 y DevExtreme. 
Muestra un inventario con búsqueda, gráfico e interacción entre componentes.

## Tecnologías
- AngularJS 1.5
- DevExtreme JS
- HTML5 / CSS3
- JavaScript

## Funcionalidades
- Grid con datos desde JSON estático
- Filtro en tiempo real por producto
- Gráfico dinámico sincronizado
- Selección de filas
- Personalización visual de inventario bajo

## Ejecución
Abrir el proyecto desde GitHub Pages o servirlo localmente con:
npx http-server .

## Estructura
- index.html
- src/app.js
- src/data.json
- styles/styles.css
